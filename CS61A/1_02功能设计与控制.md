#文档规范
函数定义通常包括描述该功能的文档， 称为docstring，必须与函数体一起缩进。 文件字符串通常用三引号。第一行描述了这份工作 在一行中完成了函数。以下几行可以描述论元和 澄清函数的行为：
>>> def pressure(v, t, n):
        """Compute the pressure in pascals of an ideal gas.

        Applies the ideal gas law: http://en.wikipedia.org/wiki/Ideal_gas_law

        v -- volume of gas, in cubic meters
        t -- absolute temperature in degrees kelvin
        n -- particles of gas
        """
        k = 1.38e-23  # Boltzmann's constant
        return n * k * t / v

#文档测试。
Python 提供了一种方便的简单测试方法 直接出现在函数的文档字符串中。文档字符串的第一行应该是 包含函数的一行描述，后面是空行。 A 接下来可能会详细描述论据和行为。此外， docstring 可能包含一个示例交互会话，调用函数：
>>> def sum_naturals(n):
        """Return the sum of the first n natural numbers.

        >>> sum_naturals(10)
        55
        >>> sum_naturals(100)
        5050
        """
        total, k = 0, 1
        while k <= n:
            total, k = total + k, k + 1
        return total
